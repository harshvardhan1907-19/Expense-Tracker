# Expense Tracker

A responsive Expense Tracker web application built using **HTML, CSS, and JavaScript** that helps users manage their income and expenses efficiently.

## Features

* Add income and expense transactions
* Real-time balance calculation
* Income, expense, and net balance summary
* Category-wise expense tracking
* Weekly and monthly transaction filtering
* Interactive spending chart using Chart.js
* Data persistence using Local Storage
* Delete transactions instantly
* Secure user input rendering with HTML escaping

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Chart.js
* Browser Local Storage API

## How It Works

1. Add a transaction with:

   * Description
   * Amount
   * Date
   * Category

2. The application automatically:

   * Updates total income
   * Updates total expenses
   * Calculates current balance
   * Stores data in Local Storage
   * Updates the spending chart

3. Users can:

   * View all transactions
   * Filter transactions by week or month
   * Delete unwanted transactions
